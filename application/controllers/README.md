# Employee Payroll Application - CodeIgniter 3

A web application based on CodeIgniter 3 to manage employee payroll data, including health deductions, employment, pension, and attendance.

## Main Features

- Management of payroll deductions (health, employment, pension, attendance)
- Login authentication
- Management of position and salary data
- Backend interface using the SB-Admin template

## Installation

1. Clone this repository.
2. Make sure PHP >= 5.3.7 and the required extensions are installed.
3. Configure the database in `application/config/database.php`.
4. Run the application through a web server (Apache/Nginx) that supports PHP.
